# Image Assets

This folder contains optimized and original image assets for the landing page.

## Structure

- `webp/` — Optimized `.webp` files (use these in production)
- `original/` — Source files (JPG/PNG) — keep for reference or future re-optimization

## Guidelines

- All new images should be converted to WebP
- Use descriptive, lowercase filenames with hyphens (e.g. `hero-emblem.webp`)
- Keep original high-resolution files in the `original/` folder
